# YouTube台本スタイルガイド
## ドラゴンクエスト10チャンネル（ビィビ × りりせ 対話形式）

サンプル台本「淡路島のドラクエアイランド編」を分析して作成したスタイルガイドです。

---

## 1. ビィビの口調・語尾パターン

### 基本的な語尾の傾向
- 語尾が柔らかく親しみやすい（「〜だよ」「〜なの」「〜から」）
- 感嘆符（！）で元気よく締める場面が多い
- 自信満々に言い切ったあと、ツッコまれるボケ担当

### 具体例（5例以上）

| パターン | 例文 | 使用場面 |
|---|---|---|
| `〜だよ` / `〜だよね` | 「それが「ドラゴンクエスト アイランド」だよね」 | 情報共有・確認 |
| `〜なの` / `〜なんだよ` | 「橋の上を走るとまるで海の上を飛んでるような感覚になるの」 | 情景・感情の描写 |
| `〜から` / `〜からね` | 「愛知県民は車の運転慣れてるから安心して」 | 理由づけ・断言 |
| `〜しちゃいました` | 「結構買っちゃいました...」 | 自白・自虐的な告白 |
| `〜て！` / `〜して！` | 「覚えておいて！」「レッツゴー！」 | 視聴者への呼びかけ・テンション上昇 |
| `〜かな〜` | 「良いんじゃないかな〜と個人的には思ってる！」 | 個人的な意見・柔らかい推薦 |
| `...` (無言・沈黙) | 「まさか尾張●牧ナンバーじゃ... 」→「...」 | ツッコまれて返す言葉がないとき |
| `そうそう！` / `うんうん！` | 「そうそう！新神戸からレンタカーを借りて」 | りりせの発言への同意・テンポアップ |

### ビィビのキャラクター特性
- ゲームが得意ではない（「画面の中でも魔王倒せてない」と認める）
- 愛知県民で車の運転に自信がある
- ショッピングでついつい買いすぎる
- 芸術センスがない（スライム色塗りをスキップ）
- フレンドがいないと思われがちだが実際はいる

---

## 2. りりせの口調・語尾パターン

### 基本的な語尾の傾向
- 疑問文が多く、ビィビに説明を引き出す「聞き役」
- ツッコミは「〜けどね」「〜じゃ...」と語尾をぼかして柔らかく刺す
- 視聴者視点で「それは〜」と共感フレーズを添える

### 具体例（5例以上）

| パターン | 例文 | 使用場面 |
|---|---|---|
| `〜けどね` | 「ビィビちゃんの場合は画面の中でも魔王倒せてないけどね」 | 柔らかいツッコミ・事実の指摘 |
| `〜だもんね` | 「子供がつまらなそうにしたらかわいそうだもんね」 | 視聴者感情への共感 |
| `〜じゃ...` (省略・探り) | 「まさか尾張●牧ナンバーじゃ...」 | 意地悪な推測・じらし |
| `〜の？` / `〜なの？` | 「ビィビちゃん運転できるの？」「ビィビちゃんはやらなかったの」 | 素朴な疑問・追及 |
| `〜だ` (断言・驚き) | 「フレンドいたんだ」 | 意外性への反応 |
| `〜かな？` | 「ちなみに事前予約は必須かな？」 | 視聴者を代表した質問 |
| `わき見運転しないでね...` | （不安を語尾「...」で表現） | 心配・釘を刺す |
| `〜は大事だからね！` | 「冒険前のエネルギーチャージは大事だからね！」 | ビィビの発言への肯定ツッコミ |

### りりせのキャラクター特性
- 視聴者が「聞きたいこと」を代わりに質問する役割
- ビィビの欠点や矛盾を優しく突っ込む
- 感情移入が豊か（「それはめっちゃ気になる...」など）
- ビィビのボケに対して冷静に一言返す

## 3. ボケとツッコミの入れ方パターン

### パターン A：ビィビ自信満々 → りりせ一刀両断

```
ビィビ:「〜だから安心して！」（根拠のない自信）
りりせ:「まさか〜じゃ...」（弱点を突く）
ビィビ:「...」（無言で認める）
```

例：
> ビィビ「愛知県民は車の運転慣れてるから安心して」
> りりせ「まさか尾張●牧ナンバーじゃ...」
> ビィビ「...」

---

### パターン B：ビィビが弱点を開示 → りりせが追撃

```
ビィビ:「〜なんだ（自虐）」
りりせ:「〜なのに〜なの？（追撃）」
ビィビ:「〜って言ってるでしょ！（言い訳）」
```

例：
> りりせ「フレンドいたんだ」
> ビィビ「いるって言ってるでしょ！」

---

### パターン C：りりせが先に心配 → ビィビが後で認める

```
りりせ:「〜しないでね...（先行して警告）」
（シーン進む）
ビィビ:「いやだ！おろして！（後で証明される）」
```

例：
> りりせ「わき見運転しないでね...」
> ビィビ「それではレッツゴー！」
> りりせ「いやだ！おろして！」

---

### パターン D：視聴者目線の共感ツッコミ

```
りりせ:「〜だもんね（視聴者と同じ気持ちを代弁）」
ビィビ:「そうそう！（肯定して次の話題へ）」
```

例：
> りりせ「遊びに行って熱中症になっちゃったら最悪だもんね」
> ビィビ「そうそう...ドラクエの主人公って〜」

---

### パターン E：ボケへの「仕方ない」許容

```
ビィビ:「〜ちゃいました（自白）」
りりせ:「まぁ〜だから仕方ないね（甘いツッコミ）」
```

例：
> ビィビ「結構買っちゃいました...」
> りりせ「まぁ旅の思い出だから仕方ないね...」

## 4. 話題の導入 → 展開 → まとめの流れ

### 全体構造

```
[冒頭の挨拶]
  └─ キャラクター名乗り + 今回のテーマ宣言

[アジェンダ提示]
  └─ 「この動画を見るとわかること」番号リスト（2〜4項目）

[フック（視聴者への問いかけ）]
  └─ 共感を呼ぶ問い → 「そんな夢が叶う場所が〜」

[本題（セクション構成）]
  アクセス方法 → 現地の食事 → チケット選び
  → アトラクション体験 → 体験コツ → グッズ
  ※各セクションは「まずは〜から！」で導入

[まとめ]
  └─ 「〜のポイントは」番号リスト（4〜6項目）
  └─ ターゲット別おすすめ締め（友達・家族・カップル）

[クロージング]
  └─ コメント欄への誘導 → 締めの挨拶
```

### セクション内の展開パターン

```
導入一文（ビィビが宣言）
  ↓
りりせの質問（視聴者の疑問を代弁）
  ↓
ビィビの説明（情報提供）
  ↓
ボケ or ツッコミ（1〜2往復）
  ↓
りりせの補足/共感（まとめ役）
  ↓
次セクションへのブリッジ
```

## 5. 視聴者への語りかけフレーズ集

### 問いかけ・共感誘導
- 「みんなはドラクエの世界に入ってみたいって思ったことない？」
- 「〜ってなったことない？」

### 情報・行動を促す
- 「覚えておいて！」
- 「気になる人は公式へ電話してね」
- 「〜がおすすめ！」
- 「〜だから安心して」
- 「〜に気をつけないとね」

### 共感・感情を引き出す
- 「〜だよ！めちゃくちゃ気になる...」
- 「〜こと間違いなしの〜なんだよ」
- 「大人から子供まで楽しめる」
- 「友達と行っても家族と行っても カップルで行っても最高の思い出になると思う」

### 体験報告への誘導（エンゲージメント）
- 「もし行ってきたよって人がいたらコメント欄で感想教えてね」
- 「ぜひ自分だけの物語を紡いでみてね」

### ネタバレ・注意の前置き
- 「この動画は私の体験談になります」
- 「若干のネタバレを含みますのでご了承ください」
- 「ネタバレにならないように公式以上の写真は載せませんが」

## 6. セクション間のつなぎ方

### 前進するつなぎ（次の話題へ移行）
- `まずは〜から！` ― 最初のセクション導入
- `次はいよいよ〜の話！` ― クライマックスへの期待感
- `最後に〜！` ― 締めくくりセクション

### 会話から自然に移行するつなぎ
- `そういえば〜だよね？` ― りりせが話題を切り替える
- `でもはやる気持ちを抑えて！` ― 逸れた話を本題に戻す
- `ちなみに〜かな？` ― 補足情報を引き出す

### 感情のピークを作るつなぎ
- `ここで重要なポイントがあります！` ― 注意喚起
- `それはめっちゃ気になる...` ― 次セクションへの興味を高める
- 「いやだ！おろして！」のような会話劇でセクション間に笑いを挟む

## 7. 冒頭の挨拶パターン

### 標準テンプレート

```
[キャラクター名乗り]
ビィビです！今回は
〔テーマ〕について
お伝えします

[アジェンダ（番号リスト）]
この動画を見るとわかることはこちらです
1. 〔ポイント1〕
2. 〔ポイント2〕
3. 〔ポイント3〕

[前置き・免責]
この動画は私の体験談になります
〔注意事項〕をご了承ください

[スタート宣言]
それでは始めていきます
```

### バリエーション要素
- テーマは「〜について」と明示
- アジェンダは2〜4項目（視聴者が内容を把握しやすいよう）
- 免責・前置きは任意（体験談・ネタバレがある場合に追加）
- スタート宣言「それでは始めていきます」は固定フレーズ

---

## 8. 締めのパターン

### 標準テンプレート

```
[ポイントまとめ（番号リスト）]
〔テーマ〕のポイントは
1. 〔ポイント1〕
2. 〔ポイント2〕
…

[ターゲット別おすすめ]
〔誰と行っても〕最高の思い出になると思う

[視聴者への問い返し・エンゲージメント]
ビィビちゃんはまた行きたい？（→ キャラクターに聞く）
すぐにでも行きたい！

[コメント欄への誘導]
もし〔体験〕した人がいたら
コメント欄で感想教えてね

[締めの言葉]
でもみんなそれぞれの〔体験〕があるから
ぜひ自分だけの物語を紡いでみてね

[クロージング定型文]
それでは本日の動画はここまでとなります
ご視聴いただきありがとうございます
```

### チャンネル登録の促し（推奨追加位置）
サンプル台本には明示的な登録促しは含まれていないが、
クロージング直前に以下を追加することを推奨：

```
チャンネル登録・高評価もよろしくね！
次回の動画もお楽しみに♪
```

## 9. 1セクションあたりの平均的なやりとり回数

### セクション別カウント（サンプル台本より）

| セクション | 発話ターン数 | 備考 |
|---|---|---|
| 冒頭の挨拶・アジェンダ | 1（ビィビ独白） | りりせ登場前 |
| フック・舞台設定 | 4〜5往復 | ドラクエの夢→淡路島の紹介 |
| アクセス方法 | 6〜8往復 | 新神戸→SA→ニジゲンノモリ |
| ルイーダの酒場 | 6〜8往復 | 外観→メニュー→チケット不要の情報 |
| チケット選び | 5〜6往復 | 3種類の説明→おすすめ→予約 |
| アトラクション体験 | 7〜9往復 | パーティ→職業→戦闘→歩く注意 |
| グッズショップ | 4〜5往復 | 種類→買いすぎ自白 |
| まとめ | 6〜8往復 | ポイントリスト→締め |

**平均：5〜8往復 / セクション**

### 会話リズムの法則
- りりせの質問1回 → ビィビの説明1〜2文 → ボケかツッコミ1往復
- 重要情報の前後に「ね！」「そうそう！」の短い相づちを挟む
- 1セクションに笑いどころ（ボケ/ツッコミ）は1〜2回

## 10. 全体の文字数と構成比率

### 文字数計測（サンプル台本）

| パート | 概算文字数 | 構成比 |
|---|---|---|
| **導入**（挨拶・アジェンダ・フック） | 約 350 字 | 約 12% |
| **本題**（アクセス・食事・チケット・体験・グッズ） | 約 2,200 字 | 約 76% |
| **まとめ**（ポイントリスト・締め・クロージング） | 約 350 字 | 約 12% |
| **合計** | **約 2,900 字** | **100%** |

### 推奨比率

```
導入 : 本題 : まとめ ≒ 1 : 6〜7 : 1
```

- 導入が長すぎると視聴者が離脱するため、アジェンダは簡潔に
- 本題は複数セクションに分割し、各セクション300〜500字程度
- まとめはポイントリスト形式で、視聴者が後で見返しやすい構成

### 読み上げ時間の目安
- 日本語は約400〜500字/分（YouTube向け適度なテンポ）
- 約2,900字 → **読み上げ約6〜7分**
- ゆっくり喋る・効果音・BGM等を含めると **動画尺10〜12分** が目安
